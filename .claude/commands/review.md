### 2. 리뷰 관점 (모두 체크)

**기능 정확성**
- 비즈니스 로직이 올바른가?
- 엣지 케이스를 처리했는가?
- 에러 핸들링이 적절한가?

**보안**
- SQL 인젝션 가능성
- XSS / CSRF 방어
- 인증/인가 누락
- 민감 정보 노출 (로그, 응답)

**성능**
- N+1 쿼리 문제
- 불필요한 DB 호출
- 비동기 처리가 적절한가 (await 누락 등)

**코드 품질**
- 타입 힌트 완전성
- 함수/변수 네이밍
- 중복 코드
- SOLID 원칙 준수

**테스트**
- 변경 사항에 대한 테스트가 있는가?
- 테스트 커버리지가 충분한가?
